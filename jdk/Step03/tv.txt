TV는 채널을 올리고 내리고 하는 기능,
음량은 음소거와 음량을 올리고 내리고 기능이 있음
전원을 켜고 끄는 기능 있으며,
채널을 올리고 내리고 음량 또한 올리고 내리고 기능 사용시 전원이 반드시 켜져있어야됨
채널 같은 경우 마지막 채널은 256이며, 처음 채널은 1입니다.
채널이 마지막 채널에서 올리면 처음 채널로 이동하고
처음 채널에서 내리면 마지막 채널로 이동합니다.
음량의 경우 최대값이 50이며, 최소값은 0입니다.
음량은 최대값 최소값에서 벗어날수 없습니다.

위에 기능들과 데이터를 가지고 있는 클래스를 작성
TV
	필드
		전원 - boolean
		채널 - int
		음량 - int
	   음소거 - boolean 
	메서드
		전원OnOff() - void
		음소거OnOff() - void
		채널Up() - void
		채널Down() - void
		음량Up() - void
		음량Down() - void
